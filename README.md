# ToDoList-V1

## About
This application is a to-do list.
Has been created using node.js, express and mongodb

## Dependencies
* Express
* Morgan
* Mongoose
* Dotenv

## Dev-Dependencies
* Nodemon

## How to install it?
To install it you just have to do a git clone and install all the dependencies by using.
```
npm install
```

## How to use it?
To use it you can run.
```
npm start
```
To start the project, or you can run.
```
npm run dev
```
To start the project using nodemon.
